Tenemos maquetado un sitio que tiene cuatro secciones algo largas y se genera mucho scroll hacia abajo. 

Queremos lograr que la barra de navegación esté siempre visible para que la navegabilidad del sitio sea óptima.

* Para eso debemos ubicar al <ul> con clase "nav-bar" de forma fija a 5px del inicio superior del body y asignarle un width del 100%.
